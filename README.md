# Weather Diary — Дневник погоды

Автор: Сайфутдинов Карим

## Описание программы

Weather Diary — это графическое приложение на Python с использованием библиотеки tkinter, позволяющее вести дневник наблюдений за погодой.

### Возможности:
- Добавление записей (дата, температура, описание, наличие осадков)
- Валидация ввода (формат даты, числовое значение температуры, непустое описание)
- Просмотр всех записей в виде таблицы
- Фильтрация записей:
- По конкретной дате
- По температуре (выше заданного значения)
- Сохранение данных в JSON-файл и загрузка из него
- Удаление выбранных записей

## Как запустить приложение

### Требования:
- Python 3.6 или выше
- Стандартные библиотеки: tkinter, json, datetime, os (входят в состав Python)
  
## Примеры использования

Для начала работы запустите приложение. Чтобы добавить первую запись о погоде, в поле «Дата» оставьте текущую дату (например, 2026-05-08), в поле «Температура» введите 18.5, в поле «Описание» напишите «Солнечно, лёгкий ветер», убедитесь что чекбокс «Есть осадки» не отмечен, и нажмите кнопку «Добавить запись». В таблице появится новая строка. Аналогично добавьте ещё несколько записей: за 7 мая с температурой 12.0 и описанием «Пасмурно, сыро» с осадками, за 6 мая с температурой 8.5 и описанием «Туман, холодно» с осадками, за 5 мая с температурой 20.0 и описанием «Жарко, солнечно» без осадков.

Теперь попробуйте фильтрацию. Чтобы увидеть погоду за конкретный день, в поле «Фильтр по дате» введите 2026-05-06 и нажмите «Применить фильтр» — в таблице останется только запись за 6 мая. Нажмите «Сбросить фильтр», чтобы вернуть все записи. Для фильтрации по температуре в поле «Температура выше» введите 10 и нажмите «Применить фильтр» — отобразятся записи с температурой выше 10°C (12.0, 18.5 и 20.0), а запись с 8.5°C скроется. Можно комбинировать фильтры: установите дату 2026-05-05 и температуру выше 15 — останется только запись от 5 мая с температурой 20.0.

Для сохранения данных нажмите кнопку «Сохранить в JSON» — в папке с программой появится файл weather_data.json. Закройте приложение и запустите снова — данные загрузятся автоматически. Также можно нажать «Загрузить из JSON» для принудительной перезагрузки.

Чтобы удалить запись, кликните на неё в таблице (строка выделится), нажмите «Удалить выбранное» и подтвердите удаление.

Программа проверяет корректность ввода. Если оставить поле «Дата» пустым, появится ошибка «Дата не может быть пустой». Если ввести дату в неверном формате (например, 32.05.2026), программа сообщит: «Неверный формат даты. Используйте ГГГГ-ММ-ДД». Если в поле «Температура» ввести текст вместо числа, вы увидите ошибку «Температура должна быть числом». Пустое описание также вызовет ошибку. При фильтрации по температуре ввод нечислового значения приведёт к сообщению «Температура фильтра должна быть числом».
